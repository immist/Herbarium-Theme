# [Herbarium](https://herbarium.co) Theme
A clean, simple and minimalist masonry WordPress theme, developed for [Herbarium](https://herbarium.co), and based on [Fukasawa](https://wordpress.org/themes/fukasawa/).

It is adapted to the use case of [Herbarium](https://herbarium.co), a free database about the health benefits and therapeutic uses of medicinal herbs, healthy plants and superfoods, but can easily be adapted for any other arbitrary use case.
